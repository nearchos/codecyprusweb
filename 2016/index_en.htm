Workshops

Select one of the workshops, and receive a cool Certificate of Participation while also learning to code!

These will be running in parallel, so each participant must select exactly one workshop.

Hour of Code

Ideal for programming novices who want to try and learn in a fun 2-hour session.

Games Challenge

Ideal for game lovers who already have some programming experience.

Ideal for anyone who wants a first taste of SQL in a fun 2-hour session.

Robotics Challenge

One of the most entertaining ways to learn programming is by programming a robot.

Treasure Hunt

Learning code is not only fun, it can also be exciting. This is why we are organizing this Treasure Hunt!

Play alone or with a team-mate. Download the Code Cyprus Android App and follow the clues that lead to the treasure.

The first six winners will each get a €200 coupon for any device purchased from Cytashops!

Please note that every person can participate in exactly one team, and only one time.

Frequently Asked Questions

What is Code Cyprus?

Code Cyprus is an annual event that aims to inspire the next generation of computer scientists. Its main goal is to 'inspire high school students to take an interest in coding'. You can learn more about the story of Code Cyprus by reading this blog post.

Is there a participation cost?

Since its inception, Code Cyprus has been free to its participants, thanks to kind sponsorships from various partners. Our aim is to keep the event free every year, and we will work hard to achieve this. Please note that despite this, we always require a registration (which is free) because we have a limited number of seats.

In what language is the event delivered?

Our aim is to keep the event as bilingual as possible: anyone who understands Greek or English should be able to take the most out of the event.

Does this include lunch?

Since Code Cyprus starts in the morning and ends in the early afternoon, we have a lunch break. This break is commonly offered by our hosting partner, UCLan Cyprus

Can I sponsor this event?

Yes. Our main goal is to deliver a highly engaged event for the benefit of our participants. Since we aim to keep this free to participating students, we always welcome kind sponsorships. If you are interested, please email us at: hello@codecyprus.org.

Who should I contact if I have another question?

If you have a question, please email us at: hello@codecyprus.org. We generally reply within 24 hours. Please do not call UCLan Cyprus.
