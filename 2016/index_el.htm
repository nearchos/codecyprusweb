5 Μαρτίου, 2016 @ UCLan Cyprus

Η ημερίδα μας ήταν η πιο πετυχημένη που διοργανώσαμε ποτέ με 133 συμμετοχές!
Διαβάστε για αυτήν στο press release και επισκεφθείτε τη σελίδα μας στο Facebook για φωτογραφίες.

Εργαστήρια

Επιλέξτε ένα από τα εργαστήρια, και λάβετε Πιστοποιητικό Συμμετοχής ενώ μαθαίνετε προγραμματισμό!

Τα εργαστήρια θα διεξάγονται παράλληλα, για αυτό πρέπει να επιλέξετε ακριβώς ένα από τα πιο κάτω.

Εισαγωγικό

Ιδανικό για όσους δεν έχουν προγραμματίσει ποτέ αλλά θέλουν να δοκιμάσουν να μάθουν σε ένα διασκεδαστικό 2-ωρο.

Παιχνίδια Υπολογιστών

Ιδανικό για όσους αγαπούν τα παιχνίδια, και έχουν ήδη κάποιες γνώσεις προγραμματισμού.

Εξερεύνηση Δεδομένων με SQL

Ιδανικό για όσους θέλουν να γνωρίσουν την γλώσσα επερωτήσεων SQL σε ένα διασκεδαστικό 2-ωρο.

Ρομποτική

Ένας από τους διασκεδαστικότερους τρόπους να μάθετε προγραμματισμό είναι να προγραμματίσετε ένα ρομπότ!

Κυνήγι Θησαυρού

Ο προγραμματισμός δεν είναι απλά ενδιαφέρον, αλλα μπορεί να είναι και διασκεδαστικός. Για αυτό διοργανώνουμε ένα συναρπαστικό Κυνήγι Θησαυρού!

Παίξτε μόνοι ή με ένα συμπαίκτη. Κατεβάστε την εφαρμογή Code Cyprus για Android και ακολουθείστε τις ενδείξεις για να φτάσετε στο θησαυρό.

Οι πρώτοι έξι νικητές κερδίζουν από ένα κουπόνι €200 ο καθένας για αγορές από τα Cytashops!

Σημειώστε ότι ο κάθε ένας μπορεί να συμμετέχει σε μόνο μια ομάδα, και μόνο μια φορά.

Συχνές Ερωτήσεις

Τι είναι το Code Cyprus;

Το Code Cyprus έιναι μια ετήσια ημερίδα που στοχεύει στο να εμπνεύσει την επόμενη γενιά επιστημόνων πληροφορικής. Ο κύριος στόχος είναι να 'ενθαρρύνει μαθητές γυμνασίων και λυκείων να μάθουν προγραμματισμό'. Μπορείτε να μάθετε περισσότερα για την ιστορία του Code Cyprus διαβάζοντας αυτό το blog post (στα Αγγλικά).

Υπάρχει κόστος συμμετοχής;

Από την πρώτη χρονιά που διοργανώθηκε, το Code Cyprus είναι δωρεάν για τους συμμετέχοντες χάρη στην ευγενή χορηγία από διάφορους συνεργάτες. Ο στόχος μας είναι να κρατήσουμε το Code Cyprus δωρεάν κάθε χρόνο, και θα δουλέψουμε σκληρά για να το πετύχουμε. Σημειώστε ότι σε κάθε περίπτωση απαιτείται δωρεάν εγγραφή, επειδή υπάρχει περιορισμένος αριθμός θέσεων.

Σε ποιά γλώσσα προσφέρεται το Code Cyprus;

Ο στόχος μας είναι να κάνουμε την ημερίδα προσβάσιμη σε δυο γλώσσες: Ελληνικά και Αγγλικά.

Περιλαμβάνεται και μεσημεριανό;

Επειδή το Code Cyprus ξεκινά το πρωί και διαρκεί μέχρι νωρίς το απόγευμα, το πρόγραμμα μας περιλαμβάνει και διάλειμμα για μεσημεριανό. Σε αυτό συνήθως προσφέρεται ελαφρύ γεύμα, προσφορά του συνεργάτη φιλοξενίας, UCLan Cyprus

Μπορώ να χορηγήσω το Code Cyprus;

Ναι. Ένας από τους βασικούς μας στόχους είναι να διοργανώνουμε την ημερίδα με τρόπο που να είναι όσο πιο ενδιαφέρον γίνεται για τους συμμετέχοντες. Επειδή η συμμετοχή είναι δωρεάν, και θέλουμε να την κρατήσουμε δωρεάν, χρειαζόμαστε κάθε βοήθεια που μπορούμε να έχουμε για την διοργάνωση. Εαν σας ενδιαφέρει να γίνεται συνεργάτης του Code Cyprus παρακαλώ επικοινωνείστε μαζί μας μέσω email στο: hello@codecyprus.org.

Πως μπορώ να επικοινωνήσω μαζί σας αν έχω κάποια άλλη ερώτηση;

Εάν έχετε κάποια ερώτηση, παρακαλώ στείλτε μας email στο: hello@codecyprus.org. Γενικά απαντούμε στις ερωτήσεις μέσω email εντός 24 ωρών. Παρακαλώ μην τηλεφωνείτε στο UCLan Cyprus για ερωτήσεις.
