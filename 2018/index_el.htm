Εργαστήρια

Επιλέξτε ένα από τα εργαστήρια, και λάβετε Πιστοποιητικό Συμμετοχής ενώ μαθαίνετε προγραμματισμό!

Τα εργαστήρια θα διεξάγονται παράλληλα, για αυτό πρέπει να επιλέξετε ακριβώς ένα από τα πιο κάτω.

Φτιάξτε την δική σας ιστοσελίδα

Ιδανικό για όσους δεν έχουν προγραμματίσει ποτέ αλλά θέλουν να δοκιμάσουν να μάθουν τα βασικά σε ένα διασκεδαστικό 2-ωρο.

Παιχνίδια Υπολογιστών

Ιδανικό για όσους αγαπούν τα παιχνίδια, και έχουν ήδη κάποιες γνώσεις προγραμματισμού.

Εξερεύνηση Δεδομένων με SQL

Ιδανικό για όσους θέλουν να γνωρίσουν την γλώσσα επερωτήσεων SQL σε ένα διασκεδαστικό 2-ωρο.

Παιχνίδι Περιπέτειας με Προγραμματισμό

Ένας από τους διασκεδαστικότερους τρόπους να μάθετε προγραμματισμό είναι μέσα από ένα 3D παιχνίδι!

Προγραμματισμός με Arduino

Μάθετε πως ο προγραμματισμός σας δίνει την δυνατότητα να ελέγχετε μηχανικά μέρη, όπως να ανάβετε λαμπάκια και να ενεργοποιείτε μοτερ!

Κυνήγι Θησαυρού

Ο προγραμματισμός δεν είναι απλά ενδιαφέρον, αλλα μπορεί να είναι και διασκεδαστικός. Για αυτό διοργανώνουμε ένα συναρπαστικό Κυνήγι Θησαυρού!

Συχνές Ερωτήσεις

Τι είναι το Code Cyprus;

Το Code Cyprus έιναι μια ετήσια ημερίδα που στοχεύει στο να εμπνεύσει την επόμενη γενιά επιστημόνων πληροφορικής. Ο κύριος στόχος του Code Cyprus είναι να ενθαρρύνει μαθητές γυμνασίων και λυκείων να μάθουν προγραμματισμό. Επιπλέον μπορείτε να δείτε διάφορες ειδήσεις και φωτογραφίες από προηγούμενες χρονιές του Code Cyprus στη σελίδα μας στο facebook. Τέλος, μπορείτε να μάθετε περισσότερα για την ιστορία του Code Cyprus διαβάζοντας αυτό το blog post (στα Αγγλικά).

Τι πρέπει να ξέρω πριν την μετάβαση μου στο UCLan Cyprus;

Το Code Cyprus διεξάγεται σε μια από τις πιο σύγχρονες πανεπιστημιούπολεις της Ευρώπης, στο UCLan Cyprus. Πριν την επίσκεψη σας, σας παρακαλούμε όπως ενημερωθείτε για τις σημάνσεις και διαδικασίες του πανεπιστημίου διαβάζοντας τον οδηγό υγείας και ασφάλειας.

Υπάρχει κόστος συμμετοχής;

Από την πρώτη χρονιά που διοργανώθηκε, το Code Cyprus είναι δωρεάν για τους συμμετέχοντες χάρη στην ευγενή χορηγία από διάφορους συνεργάτες. Ο στόχος μας είναι να κρατήσουμε το Code Cyprus δωρεάν κάθε χρόνο, και θα δουλέψουμε σκληρά για να το πετύχουμε. Σημειώστε ότι σε κάθε περίπτωση απαιτείται δωρεάν εγγραφή, επειδή υπάρχει περιορισμένος αριθμός θέσεων.

Σε ποιά γλώσσα προσφέρεται το Code Cyprus;

Ο στόχος μας είναι να κάνουμε την ημερίδα προσβάσιμη σε δυο γλώσσες: Ελληνικά και Αγγλικά.

Τι χρειάζομαι για να συμμετέχω στο Κυνήγι Θησαυρού;

Απαιτείται η χρήση μια κινητής συσκευής (smartphone ή tablet). Ιδανικά θα έχετε μια μοντέρνα Android συσκευή για να χρησιμοποιήσετε την εφαρμογή μας. Αλλά αν έχετε κάποια άλλη συσκευή (π.χ. iPhone, iPad, Windows phone, κλπ) μπορείτε να χρησιμοποιήσετε το Web App μας. Σε κάθε περίπτωση, βεβαιωθείτε πως έχετε ενεργοποιημένο το GPS σας.

Περιλαμβάνεται και μεσημεριανό;

Επειδή το Code Cyprus ξεκινά το πρωί και διαρκεί μέχρι νωρίς το απόγευμα, το πρόγραμμα μας περιλαμβάνει και διάλειμμα για μεσημεριανό. Σε αυτό συνήθως προσφέρεται ελαφρύ γεύμα, προσφορά του συνεργάτη και συνδιοργανωτή, UCLan Cyprus.

Μπορώ να χορηγήσω το Code Cyprus;

Ναι. Ένας από τους βασικούς μας στόχους είναι να διοργανώνουμε την ημερίδα με τρόπο που να είναι όσο πιο ενδιαφέρον γίνεται για τους συμμετέχοντες. Επειδή η συμμετοχή είναι δωρεάν, και θέλουμε να την κρατήσουμε δωρεάν, χρειαζόμαστε κάθε βοήθεια που μπορούμε να έχουμε για την διοργάνωση. Εαν σας ενδιαφέρει να γίνετε συνεργάτης του Code Cyprus παρακαλώ επικοινωνείστε μαζί μας μέσω email στο: hello@codecyprus.org.

Πως μπορώ να επικοινωνήσω μαζί σας αν έχω κάποια άλλη ερώτηση;

Εάν έχετε κάποια ερώτηση, παρακαλώ στείλτε μας email στο: hello@codecyprus.org. Γενικά απαντούμε στις ερωτήσεις μέσω email εντός 24 ωρών. Παρακαλώ μην τηλεφωνείτε στο UCLan Cyprus για ερωτήσεις.
