Δωρεάν εισιτήριο

Είστε μαθητής γυμνασίου ή λυκείου; Ελάτε να περάσουμε μια καταπληκτική μέρα, φτιάχνοντας το δικό σας Frogger παιχνίδι για κινητά, κερδίστε σημαντικά βραβεία σε ένα διασκεδαστικό Κυνήγι Θησαυρού χρησιμοποιώντας το smart-phone σας, και εμπνευστείτε από ένα ντοκιμαντέρ για νέους web entrepreneurs στην Ευρώπη και Αμερική. Πάρτε το δωρεάν εισιτήριο σας τώρα.

Τα βραβεία που μπορείτε να κερδίσετε στο Κυνήγι Θησαυρού

Γίνετε οι πρώτοι που θα φορέσετε στα χέρια σας ένα από τα πιο έξυπνα, και διασκεδαστικά προγραμματιζόμενά ρολόγια που έχουν κατασκευαστεί ποτέ.

Οι πρώτες τρεις ομάδες θα κερδίσουν από ένα μοναδικό Pebble smartwatch για κάθε μέλος της ομάδας.

Πάρτε μέρος στο πιο διασκεδαστικό και high-tech Κυνήγι θησαυρού στην Κύπρο! Πάρτε το Android κινητό σας και λύστε τους γρίφους για να βρείτε τον θησαυρό. Οι τρείς ομάδες με το ψηλότερο σκορ θα κερδίσουν βραβεία.

Παρακολουθείστε μαζί μας το The startup kids, ένα ντοκιμαντέρ για νεαρούς web entrepreneurs σε Ευρώπη και Αμερική. Το αμφιθέατρο του UCLan Cyprus, που βραβεύθηκε για την οπτικο-ακουστική του τεχνολογία, εγγυάται μια αξέχαστη εμπειρία.
