March 7th, 2015 @ UCLan Cyprus

Code Cyprus 2015 was a big hit! Already looking forward to our next year's event :-)

Make sure to like our Facebook page to keep updated with our latest news, including many photos from the event.

Workshops

Select one of the workshops, and receive a cool Certificate of Participation while also learning to code!

These will be running in parallel, so each participant must select exactly one workshop.

Hour of Code

Ideal for programming novices who want to try and learn in a fun 2-hour session.

Games Challenge

Ideal for game lovers who already have some programming experience.

Querying Data with SQL

Ideal for anyone who wants a first taste of SQL in a fun 2-hour session.

Robotics Challenge

For those who are not very skilled in programming but would like to try and learn how to program a robot.

Treasure Hunt

Learning code is not only fun, it can also be exciting. This is why we are organizing this Treasure Hunt!

Play alone or with a team-mate. Download the Code Cyprus Android App and follow the clues that lead to the treasure.

The first six winners will each get a €200 coupon for any device purchased from Cytashops!

Please note that every person can participate in exactly one team, and only one time.
