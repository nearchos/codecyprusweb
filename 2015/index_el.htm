7 Μαρτίου, 2015 @ UCLan Cyprus

Το Code Cyprus 2015 ήταν επιτυχία! Ανυπομονούμε ήδη για την επόμενη χρονιά :-)

Μην ξεχάσετε να κάνετε like την σελίδα μας στο Facebook για να ενημερώνεστε για τα νέα μας, αλλά και για πολλές φωτογραφίες.

Εργαστήρια

Επιλέξτε ένα από τα εργαστήρια, και λάβετε Πιστοποιητικό Συμμετοχής ενώ μαθαίνετε προγραμματισμό!

Τα εργαστήρια θα διεξάγονται παράλληλα, για αυτό πρέπει να επιλέξετε ακριβώς ένα από τα πιο κάτω.

Εισαγωγικό

Ιδανικό για όσους δεν έχουν προγραμματίσει ποτέ αλλά θέλουν να δοκιμάσουν να μάθουν σε ένα διασκεδαστικό 2-ωρο.

Παιχνίδια Υπολογιστών

Ιδανικό για όσους αγαπούν τα παιχνίδια, και έχουν ήδη κάποιες γνώσεις προγραμματισμού.

Εξερεύνηση Δεδομένων με SQL

Ιδανικό για όσους θέλουν να γνωρίσουν την γλώσσα επερωτήσεων SQL σε ένα διασκεδαστικό 2-ωρο.

Ρομποτική

Για όσους δεν γνωρίζουν καλά προγραμματισμό αλλά θα ήθελαν να δοκιμάσουν να μάθουν και να προγραμματίσουν ένα ρομπότ.

Κυνήγι Θησαυρού

Ο προγραμματισμός δεν είναι απλά ενδιαφέρον, αλλα μπορεί να είναι και διασκεδαστικός. Για αυτό διοργανώνουμε ένα συναρπαστικό Κυνήγι Θησαυρού!

Παίξτε μόνοι ή με ένα συμπαίκτη. Κατεβάστε την εφαρμογή Code Cyprus για Android και ακολουθείστε τις ενδείξεις για να φτάσετε στο θησαυρό.

Οι πρώτοι έξι νικητές κερδίζουν από ένα κουπόνι €200 ο καθένας για αγορές από τα Cytashops!

Σημειώστε ότι ο κάθε ένας μπορεί να συμμετέχει σε μόνο μια ομάδα, και μόνο μια φορά.
