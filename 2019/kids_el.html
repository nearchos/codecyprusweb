Code Cyprus 2019

Robot Adventures

Για παιδιά — ηλικίας 4 ή 5 ετών — με τους γονείς τους



Αυτή τη χρονιά προσκαλούμε γονείς με τα παιδιά τους να δοκιμάσουν το Robot Adventures, ένα καινοτόμο πρόγραμμα ρομποτικής.

Το Robot Adventures συνδυάζει την περιπέτεια και το παιχνίδι μέσα από τον προγραμματισμό καλλιεργώντας την κριτική σκέψη των παιδιών και την ικανότητα επίλυσης προβλημάτων.
GRASSROOTS
Με τους Δρ Παναγιώτη Ανδρέου και Δρ Ειρήνη Πολυκάρπου και Δρ Ιλιάνα Σταύρου

Κάθε παιδί πρέπει να συνοδεύεται από ένα γονιό ή κηδεμόνα για να βοηθά με το εργαστήριο. Κάθε γονιός ή κηδεμόνας μπορεί να συνοδεύσει ένα ή δύο παιδιά.
Αυτό το 45-λεπτο εργαστήριο τρέχει ανεξάρτητα από το υπόλοιπο πρόγραμμα του Code Cyprus. Διεξάγεται στις 9:30 - 10:15 στο δωμάτιο 018. Παρακαλώ προγραμματείστε να φθάσετε 15 λεπτά πριν την έναρξη του εργαστηρίου για να συμπληρώσετε την εγγραφή σας.
Όπως και το υπόλοιπο πρόγραμμα του Code Cyprus, αυτό το εργαστήριο προσφέρεται δωρεάν.
Δωρεάν Εισιτήριο!
