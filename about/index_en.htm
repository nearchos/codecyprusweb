Mission

Back in early October 2013, the Cyprus Fulbright Commission announced the Alumni Small Grants Competition, in which a group of alumni could be awarded funding for implementing a project that benefits the island as a whole. With a small team that shares the same vision, we applied for an event where the main aim was simple:

Code Cyprus will organize fun, entertaining and educating events with a goal of promoting the value and beauty of computer programming to teenagers in Cyprus and beyond.

Interested to learn more about Code Cyprus and its story? Have a look at this blog post open_in_new.

Past Events

Code Cyprus 2014 8th of March 2014

Code Cyprus 2014 was the first large scale event taking place in Cyprus, with the simple aim of Inspiring high school students to take an interest in computer programming.

Code Cyprus 2015 7th of March 2015

First Code Cyprus to run parallel workshops (covering Introductory programming, Computer games, Discovering data with SQL, and Robotics).

Code Cyprus 2016 5th of March 2016

The largest event at the time, with more than 130 students hunting for the treasure. A record that we should break again soon :-)

Code Cyprus 2017 4th of March 2017

Building on its reputation, Code Cyprus 2017 attracted many new students, and introduced a new workshop on Programming for Arduino.

Code Cyprus 2018 3rd of March 2018

Code Cyprus 2018 was another successful event with more than a hundred students attending.

Code Cyprus 2019 2nd of March 2019

Another good year for Code Cyprus with many happy students attending and enjoying a day of coding.

Code Cyprus 2020 7th of March 2020

Code Cyprus took its participants to an exciting journey in space with the power of coding.

Code Cyprus 2021 Postponed because of Covid-19

We will be back in 2022!

euro_symbol   Can I sponsor Code Cyprus?

Yes. Our main goal is to deliver a highly engaging event for the benefit of our participants. Since we aim to keep this free to participating students, we always welcome kind sponsorships. If you are interested, please email us at hello@codecyprus.org open_in_new.
