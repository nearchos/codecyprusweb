Αποστολή

Στις αρχές του Οκτώβρη του 2013, το Cyprus Fulbright Commission ανακοίνωσε ένα διαγωνισμό για Μικρές Χορηγίες Αποφοίτων, στον οποίο ομάδες από αποφοίτους μπορούσε να εξασφαλίσει χορηγία για την υλοποίηση ενός έργου που να οφελεί ολόκληρη την Κύπρο. Με μια μικρή ομάδα που μοιράζεται το ίδιο όραμα, προτείναμε μια ημερίδα με ένα απλό στόχο:

Το Code Cyprus θα οργανώνει διασκεδαστικές, ψυχαγωγικές και εκαπιδευτικές ημερίδες με στόχο την ανάδειξη της αξίας και ομορφιάς του προγραμματισμού υπολογιστών σε νέους στην Κύπρο και αλλού.

Ενδιαφέρεστε να μάθετε περισσότερα για το Code Cyprus και την ιστορία του; Ρίξτε μια ματιά σε αυτό το blog post open_in_new.

Προηγούμενες ημερίδες

Code Cyprus 2014 8 Μαρτίου 2014

Το Code Cyprus 2014 ήταν η πρώτη ημερίδα του είδους που διοργανώθηκε στην Κύπρο, με τον απλό στόχο Να ενθαρρύνει μαθητές γυμνασίων και λυκείων να ενδιαφερθούν για τον προγραμματισμό υπολογιστών.

Code Cyprus 2015 7 Μαρτίου 2015

Το πρώτο Code Cyprus που εισήγαγε παράλληλα εργαστήρια (καλύπτωντας Εισαγωγή σε προγραμματισμό, Παιχνίδια υπολογιστών, Ανακαλύπτωντας δεδομένα με SQL, και Ρομποτική).

Code Cyprus 2016 5 Μαρτίου 2016

Η μεγαλύτερη ημερίδα για την ώρα, με περισσότερους από 130 μαθητές να κυνηγούν τον θησαυρό. Ένα ρεκορ που πρέπει να ξεπεράσουμε σύντομα :-)

Code Cyprus 2017 4 Μαρτίου 2017

Βασισμένο πλεον στην φήμη του, το Code Cyprus 2017 δέχτηκε πολλούς νέους μαθητές, και εισήγαγε ένα νέο εργαστήριο για Προγραμματισμό με Arduino.

Code Cyprus 2018 3 Μαρτίου 2018

Το Code Cyprus 2018 ήταν ακόμη μια επιτυχημένη ημερίδα, με περισσότερους από 100 συμμετέχοντες!

Code Cyprus 2019 2 Μαρτίου 2019

Ακόμη μια καλή χρονιά για το Code Cyprus με πολλές συμμετοχές και κέφι για μια μέρα με κώδικα.

Code Cyprus 2020 7 Μαρτίου 2020

Το Code Cyprus μετέφερε τους συμμετέχοντες σε ένα φανταστικό ταξίδι στο διάστημα με την δύναμη του κώδικα.

Code Cyprus 2021 Αναβλήθηκε λόγω Covid-19
